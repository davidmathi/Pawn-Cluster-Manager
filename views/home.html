About PAWN

Applications can be easily sandboxed in minutes compared to days that VM configuration would take. Docker is one of the major players in this field and Kuberenetes is an orchestration tool which is used to distribute docker containers, manage and monitor the cluster of hosts

Monitoring tool for docker containers is necessary for the system administrators to take actions quickly based on the resource usage. Actions like scaling in and scaling out needs to be taken quickly when there is a surge in network, memory or CPU usage. These resources needs to be monitored frequently in order to make such decision.

So PAWN The tool will present a GUI for easy monitoring of the resources used by containers in a host. It would also be able to aggregate all the host resource utilization of each cluster. Grouping can also be done based on the pods in cluster.
